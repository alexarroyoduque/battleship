# Battleship - Hundir la flota

My first experiment with [Angular 2](https://angular.io/) is a Battleship game.


## Development

### Install dependencies

```sh
$ npm install
```

```sh
$ npm run postinstall
```

### Run project

```sh
$ npm start
```
